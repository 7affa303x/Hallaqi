# قائمة تحقق الإطلاق — Hallaqi

استخدم قبل كل نشر للإنتاج.

## تقنية
- [ ] `npm run typecheck` و `npm run test:run` و `npm run build` ناجحة
- [ ] `npm run audit` بدون ثغرات حرجة غير مقبولة
- [ ] متغيرات البيئة: Supabase، SITE_URL، مفاتيح AI، VAPID إن لزم
- [ ] `/api/health` و `/status` يستجيبان
- [ ] Sitemap و robots يشيران إلى `hallaqi.app`

## منتج
- [ ] الحجز النقدي يعمل من البداية للنهاية
- [ ] سياسة الإلغاء ظاهرة في خطوات الحجز
- [ ] راية التحليلات تظهر مرة واحدة فقط
- [ ] لا بيانات تجريبية ظاهرة في الاكتشاف
- [ ] Visit Store يطلب تأكيداً قبل المغادرة
- [ ] Stripe/CCP مخفيان أو متوقفان في الجزائر عند الإطلاق

## أمن وخصوصية
- [ ] سياسة الخصوصية محدّثة وسهلة الوصول
- [ ] حذف الحساب وتصدير البيانات يعملان

## ملاحظة حدود المعدّل
